# Composition (React + TS)

Exemplo simples de **composition** no React usando **compound components** para montar um componente de mensagem de chat (`Message`) com partes independentes.

## Tecnologias

- React + TypeScript
- Vite
- (Opcional) TailwindCSS para classes utilitárias

## Instalação

```bash
pnpm install
# ou
npm install
# ou
yarn
```
